# Momentum

크롬앱 Momentum 클론을 통해 JS를 연습을 합니다. 
## 기능
<ul>
 <li>로컬 스토리지 이용한 Greeting, To do list</li>
 <li>날씨 API를 통한 Weather</li>
 <li>Date 함수를 이용한 Clock</li>
 <li>Random 함수를 이용한 Random Image</li>
</ul>
